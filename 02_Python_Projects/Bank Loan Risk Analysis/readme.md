# 💳 Bank Loan Risk Analysis

## 📌 Project Overview

This project analyzes bank loan applicant data to identify patterns of loan default and assess financial risk.

The objective is to help financial institutions reduce potential losses by understanding borrower behavior and improving loan approval strategies using data-driven insights.

---

## 🎯 Business Problem

Banks face financial losses due to loan defaults.  
The organization wants to:

- Identify high-risk borrowers  
- Analyze default patterns  
- Evaluate impact of credit score on loan repayment  
- Reduce overall financial exposure  

As a Data Analyst, the goal is to transform raw loan data into actionable insights.

## 🔍 Key Analysis Performed

### ✅ 1. Data Cleaning & Preparation
- Checked for missing values  
- Converted default status into binary format  
- Structured credit score buckets for risk classification  

---

### ✅ 2. Default Rate Analysis
- Overall loan default percentage  
- Default rate by gender  
- Default rate by marital status  
- Default rate across credit score ranges  

---

### ✅ 3. Credit Risk Segmentation

Credit score categorized into:

- **High Risk** (550–650)  
- **Medium Risk** (651–750)  
- **Low Risk** (751+)  

Analyzed default distribution across these risk categories.

---

### ✅ 4. Financial Exposure Metrics

- Total loan amount issued  
- Total defaulted loan value  
- Estimated loss percentage  
- Income comparison between defaulters and non-defaulters  

---

### ✅ 5. Data Visualization

- Default rate by credit score bucket  
- Default distribution by gender  
- Income vs Loan Amount scatter analysis  
- Loan default pie distribution  

---

## 📊 Key Insights

- Lower credit scores show significantly higher default probability  
- High-risk borrowers contribute disproportionately to financial loss  
- Income level impacts repayment behavior  
- Certain demographic segments show higher default trends  

---

## 💡 Business Recommendations

- Tighten approval criteria for high-risk credit score groups  
- Adjust loan limits based on income-to-loan ratio  
- Implement risk-based interest pricing  
- Strengthen monitoring for medium-risk borrowers
